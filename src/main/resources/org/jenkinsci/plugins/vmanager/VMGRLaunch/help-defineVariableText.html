In case of a need in setting the vsif define values, you can use this text area for giving a list of relevant defines and their values (one line per each attribute) in the following format.
Line 1: define_name,define_value
Line 2: define_name,define_value

Example:
REG_MODE,sanity
SESSION_NAME,my_session
 (Note the comma separator within the lines. Do not place comma at the end of the lines.)
